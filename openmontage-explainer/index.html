Workflow

Agent-First Design

# Your AI assistant IS the orchestrator
You: "Make an explainer about black holes"
  ↓ Agent reads pipeline manifest (YAML)
  ↓ Agent reads director skills (Markdown)
  ↓ Agent calls Python tools with scored selection
  ↓ Agent self-reviews against quality gates
  ↓ Agent checkpoints state (resumable)
  ↓ You approve → Render → Final video
